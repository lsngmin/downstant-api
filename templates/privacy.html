개인정보 처리방침

개발자 SEUNGMIN LEE(이하 '개발자')는 사용자의 개인정보를 소중히 다루며, 관련 법령을 준수합니다. 본 방침은 앱 내 서비스 제공 과정에서 발생하는 개인정보 처리에 대해 설명합니다.

본 앱은 Vercel 및 Supabase 인프라를 사용하여 데이터를 안전하게 처리하고 있습니다.

1. 수집하는 개인정보 항목 및 방법

개발자는 앱 내 기능을 제공하기 위해 아래와 같은 정보를 수집합니다.

  • 미디어 추출 기능 (/extract): 사용자가 입력한 URL 정보를 처리합니다. (서버에 저장하지 않으며 추출 목적으로만 휘발성으로 사용됩니다.)
  • 문의하기 기능 (/api/v1/contact): 사용자 아이디, 기기 모델명, OS 버전, 문의 내용.

2. 개인정보의 수집 및 이용 목적

  • 서비스 제공: 트위터 등 외부 미디어 URL에서 다운로드 가능한 주소를 추출하여 제공.
  • 고객 응대: 사용자가 보낸 문의 사항을 확인하고 피드백을 제공하며 서비스 장애를 해결하기 위함.

3. 개인정보의 보유 및 이용 기간

수집된 문의 데이터는 원칙적으로 1년 동안 보관 후 파기됩니다. 단, 사용자가 파기를 요청할 경우 지체 없이 삭제합니다.

4. 개인정보의 제3자 제공

개발자는 사용자의 동의 없이 개인정보를 제3자에게 제공하지 않습니다. 다만, 서비스 운영을 위해 아래의 플랫폼을 활용합니다.

  • Vercel: 서버 호스팅 및 API 엔드포인트 운영
  • Supabase: 문의 데이터의 안전한 저장 (PostgreSQL 기반)

5. 사용자의 권리

사용자는 언제든지 본인이 제출한 정보의 삭제를 요청할 수 있습니다. 문의 사항이 있을 경우 앱 내 '문의하기' 기능을 이용해 주시기 바랍니다.
